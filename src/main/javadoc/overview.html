The default value is "true". @sfera.config idle_timeout number of seconds of inactivity on an interface after which the user is logged out @sfera.config manual_rebuild boolean value: "true" to disable automatic build of interfaces when webapp files are modified. The default value is "false". @sfera.config js_use_min boolean value: "true" to load the minified version of the JavaScript files of the Web App (i.e. '.min.js' extension), "false" to use the normal version (i.e. '.js' extension). The default value is "true".
